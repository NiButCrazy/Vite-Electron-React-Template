# Vite + Electron + React + TypeScript 创建模板
个人自用

### 第三方库
- Tailwind CSS - 基础样式表
- React Router - 路由管理
- Zustand - 状态管理

## 项目配置

### 安装依赖

```bash
$ yarn
```

### 运行调试

```bash
$ yarn dev
```

### 构建发布

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
